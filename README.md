# Proyecto Final Telematica 

Juego tipo Pacman desarrollado con:

- Flask
- Docker
- AWS EC2
- HTML5
- CSS
- JavaScript

---

# Características

- Movimiento de Pacman
- Fantasma enemigo
- Sistema de puntaje
- Game Over
- Sistema de victoria
- Mapa tipo arcade
- Contenedor Docker
- Despliegue en AWS

---

# Requisitos

Instalar:

- Docker Desktop
- Git

---

# Clonar el repositorio

```bash
git clone https://github.com/rodriguezmosqueraluisleonardo-wq/Proyecto-Final-Telem-tica-.git
```

---

# Entrar al proyecto

```bash
cd Proyecto-Final-Telematica
```

---

# Construir la imagen Docker

```bash
docker build -t pacman .
```

---

# Ejecutar el contenedor

```bash
docker run -d -p 8005:8005 pacman
```

---

# Abrir el juego

Abrir en el navegador:

```text
http://localhost:8005
```
